## 一、如何开启 Vultr API 功能

1. 登录 Vultr 官网后台
2. 点击左侧导航栏的「Account」选项
3. 选择「API」子菜单
4. 找到并点击「Enable API」按钮即可完成开启

API 功能开启后，您将获得专属的 API Key，这是调用所有接口的必要凭证。
